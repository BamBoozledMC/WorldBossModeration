Other Projects

These are some other similar projects I am also currently working on.

Age of Darkness Bot

The Age of Darkness Discord bot is a simple and easy to use suggestion bot. The suggestion system is almost exactly the same as the one in World Boss Moderation, with minor changes to suit the different workspace.
This bot also has a similar Linking system that includes the user's game statistics in the AoD game.

Purchase Age of Darkness on Steam.

BeansNFT Bot

The BeansNFT Discord Bot is a more complex, easy to use bot. The suggestion system includes 2 suggsetion channels, one for all users, and the other one for members with a certain role. This bot also includes a new questions system which has the same framework as suggestions, just changed to questions.
BeansNFT Bot also does not have a linking system, as it is not needed.
